# Configurações

## Visão geral

O Adobe Learning Manager fornece configurações detalhadas de administrador que permitem configurar e personalizar o sistema de gerenciamento de aprendizagem para atender às necessidades específicas da sua organização. As configurações estão organizadas em várias categorias principais, cada uma servindo funções administrativas distintas.

### Configurações básicas

* Esta seção permite que os administradores configurem detalhes organizacionais essenciais, incluindo país/região, fuso horário ou localidade.
* Configurar mecanismos de feedback e sistemas de classificação para objetos de aprendizado.
* Configurações gerais diversas que afetam o comportamento e o uso geral da plataforma.

Consulte [configurações](/help/migrated/administrators/feature-summary/settings/basic-settings.md) no Adobe Learning Manager para obter mais informações.

### Configurações de integração

* Configure as opções de autenticação, incluindo Adobe ID e logon único (SSO).
* Gerenciar integrações com sistemas de dados externos para provisionamento de usuários, catálogos de cursos e sincronização de dados de relatórios.
* Configure e gerencie conexões com outras contas do Learning Manager para o compartilhamento de conteúdo e iniciativas de aprendizado colaborativas.

Consulte as configurações no Adobe Learning Manager para obter mais informações.

### Configurações avançadas

* Crie e gerencie rótulos personalizados para organizar e categorizar o conteúdo de aprendizado, facilitando para os alunos a localização de materiais relevantes.
* Configurar e gerenciar locais físicos de sala de aula para sessões de treinamento ministradas por instrutor, incluindo detalhes de capacidade, equipamento e agendamento.
* Configure os painéis de controle Conformidade e Sucesso do grupo.

Consulte as configurações no Adobe Learning Manager para obter mais informações.
