# Duplicar e reutilizar um relatório no Report Builder

A duplicação de um relatório copia todas as suas colunas, aliases, grupos por configurações, agregações, filtros e classificação em um novo relatório editável. Use esta opção quando quiser criar uma versão de um relatório para um catálogo, grupo de usuários ou período de tempo diferente.

## Duplicar um relatório

1. Abra o relatório que você deseja duplicar.
2. Selecione **Ações** > **Duplicar**. Uma cópia do relatório é aberta no modo de edição com o nome “[nome do relatório original]-Copy”. Por exemplo, exibição de desempenho estruturado entre gerentes - Copiar
3. Informe um novo nome para o relatório.
4. Faça todas as alterações, por exemplo, atualizar filtros, ajustar colunas ou alterar a classificação.
5. Selecione **Salvar Relatório**.

O relatório duplicado é salvo na guia **Relatórios** e é independente do original.

## Duplicar um modelo

Você também pode duplicar modelos da guia **Modelos**. A duplicação de um modelo cria um novo relatório editável na guia **Relatórios**. O modelo original permanece inalterado.

1. Selecione a guia **Modelos**.
2. Localize o modelo que você deseja copiar.
3. Selecione **Duplicar**.
   ![](assets/image049.png)
4. Insira um nome, faça ajustes e selecione **Salvar Relatório**.

## Práticas recomendadas

* Duplique um relatório bem testado como base quando precisar de variações para diferentes equipes, catálogos ou períodos de tempo. Isso é mais rápido do que criar do zero e garante uma estrutura consistente.
* Renomeie a cópia imediatamente para refletir o que a torna diferente do original; por exemplo, “Conformidade por VP - APAC” em vez de “Cópia do relatório de conformidade”.
* Depois de duplicar um modelo, atualize o nome antes de fazer qualquer outra alteração para que o relatório seja identificável na guia **Relatórios**.
