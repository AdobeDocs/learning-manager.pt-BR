# Report Builder no Adobe Learning Manager

Crie e baixe relatórios personalizados com as colunas, filtros e dados que escolher, sem pós-processamento usando qualquer ferramenta.

O Report Builder da Adobe Learning Manager oferece aos administradores uma tela de relatórios de autoatendimento para criar exatamente os relatórios de que precisam. Em vez de baixar um relatório fixo e remodelá-lo em uma ferramenta de planilha, selecione as colunas desejadas, aplique filtros e baixe uma saída limpa, tudo de um só lugar.

## Por que o Report Builder foi introduzido

Os relatórios existentes no Adobe Learning Manager foram corrigidos. Cada relatório tem uma lista definida de colunas que você não pode alterar, e as opções de filtragem são limitadas. Os administradores que precisavam de saída personalizada tinham que baixar vários relatórios, uni-los ao Excel, aplicar filtros manualmente e repetir o processo toda semana ou mês.

O Report Builder remove esse ciclo. Configure um relatório uma vez, salve-o e baixe dados novos sempre que precisar, sem precisar remodelar.

Três problemas específicos que o Report Builder resolve:

* **Colunas fixas:** os relatórios existentes incluem muitas colunas que talvez não sejam necessárias e excluem as que você precisa. Report Builder permite escolher apenas os campos relevantes para o caso de uso.
* **Filtragem limitada:** agora você pode filtrar qualquer campo compatível. Por exemplo, incluindo a data de inscrição, a data de conclusão, os campos ativos e as etiquetas de catálogo, não apenas o status.
* **Fontes de dados inconsistentes:** os relatórios existentes podem ser extraídos da interface do usuário, da API de Trabalho ou dos conectores, que podem retornar dados ligeiramente diferentes dependendo do tempo. O Report Builder obtém de uma única fonte de dados consistente, de modo que cada download reflete os mesmos dados subjacentes.

## Como o Report Builder se adapta aos relatórios existentes

O Report Builder não substitui os relatórios fixos existentes no Adobe Learning Manager. Ambos estão disponíveis. Use os relatórios existentes para saídas padrão nas quais você já confia. Use Report Builder quando precisar de colunas personalizadas, filtros específicos ou dados que atualmente exigem pós-processamento.

## Onde encontrar o Report Builder

O Report Builder está disponível na página inicial do administrador. Selecione **Relatórios**, depois selecione **Report Builder**. Você verá duas guias: **Modelos** e **Relatórios**.
![](assets/report-builder-001.png)

## Quem pode usar o Report Builder

O Report Builder está disponível para administradores.

>[!NOTE]
>
>O Report Builder precisa ser ativado para contas. Entre em contato com a equipe da sua conta de Adobe ou com o CSM se não for exibido no modo de exibição de administrador.

## Práticas recomendadas

* Comece com um modelo, se for novo no Report Builder. Os modelos são pré-criados para os casos de uso mais comuns e funcionam imediatamente.
* Salve todos os relatórios configurados antes de baixar. Os relatórios salvos podem ser baixados novamente sem reconfiguração.
